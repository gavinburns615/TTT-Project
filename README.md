I made a tic tac toe game that uses the users inputs through the terminal to select the X and O location. 
In addition to the normal requirments of the project I also included a score board where it would keep tack
of winss for Xs and Os and the amount of draws. This led to me also adding a play again feature for the user.
The project was split up into 5 main methods. The first being print board. This methods upon being called in
the main() function printed the classic tic tac toe board. The next method was check winner method. When being
called this owuld check for straight and diagonal wins. The third method checks to see if the board is all
filled up and in that case it is a draw. Another smaller method I used was printing the score so you could 
see how many times each player has won loss and tied. To implement them all the main method takes the input
from the user from the terminal and calls all the other methods to make it the deseired output for the tic
tac toe game
